Data Migration

Important

  • Migration will copy all data from SQLite to the external database
  • The original SQLite file will be preserved as a backup
  • Test the connection before migrating
  • Large datasets may take several minutes to migrate

Security Recommendations

  • SSL/TLS: Always enable encryption for production databases
  • IP Whitelist: Restrict database access to only the VAAS server IP
  • Least Privilege: Create a dedicated database user with only necessary permissions
  • Private Network: Use VPC/VNET for cloud databases when possible
  • Strong Passwords: Use complex passwords with at least 16 characters
